# Factory Api
This was the coolest API build I've ever done! I learned about:

* [API Documentation](https://documenter.getpostman.com/view/23374730/2s8YY9v6YQ) 
* Active Model Serializer
* Active Record Associations
* State Machine

### Setup Versions
* Ruby version: 3.0.4
* Rails version: 6.1.7

##### Install gems
```sh
$ bundle install
```
##### Database migration
```sh
$ rails db:migrate 
```
##### Populate database
```sh
$ rails dev:setup
```
